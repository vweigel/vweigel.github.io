Project 1: Simple Physics Simulation

This project simulates the motion of a projectile under gravity.

Code:

# Simple projectile motion simulator
import math

# Initial conditions
v0 = 20  # initial velocity in m/s
angle = 45  # launch angle in degrees
g = 9.81  # gravitational acceleration in m/s^2

# Convert angle to radians
angle_rad = math.radians(angle)

# Calculate initial velocity components
v0_x = v0 * math.cos(angle_rad)
v0_y = v0 * math.sin(angle_rad)

# Calculate flight time and range
flight_time = (2 * v0_y) / g
range_ = v0_x * flight_time

# Display results
print(f"Flight Time: {flight_time:.2f} seconds")
print(f"Range: {range_:.2f} meters")

Project 2: Linear Regression in Python

This project fits a line to a given data set using linear regression.

Code:

import numpy as np
import matplotlib.pyplot as plt

# Sample data points
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 5, 4, 5])

# Linear regression calculation
A = np.vstack([x, np.ones_like(x)]).T
m, c = np.linalg.lstsq(A, y, rcond=None)[0]

# Plot the data and the fitted line
plt.scatter(x, y, color='red', label='Data points')
plt.plot(x, m*x + c, 'b', label=f'Fitted line: y = {m:.2f}x + {c:.2f}')
plt.legend()
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Linear Regression')
plt.show()
